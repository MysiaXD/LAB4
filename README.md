## 📝 Опис варіанту

1. **Базове завдання (дія з текстом):** Знайти таке слово в першому реченні заданого тексту, якого немає в жодному з наступних.
2. **Архітектурні вимоги (композиція класів):** - Замінити послідовність табуляцій та пробілів одним пробілом.
   - Створити окремі класи для складових тексту: `Letter` (літера), `Word` (слово), `Punctuation` (розділовий знак), `Sentence` (речення) та `Text` (текст).
   - Забезпечити ієрархію: Слово складається з масиву літер; Речення — з масиву слів та розділових знаків; Текст — з масиву речень.
